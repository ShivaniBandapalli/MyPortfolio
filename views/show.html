 Image Steganography using Convolutional Neural Networks (CNN) Basedd Encoder-Decoder Model

A deep learning-based image steganography system using a CNN-based encoder-decoder architecture. It enables hiding and retrieving secret messages or images within a cover image, making communication secure and invisible to intruders.

GitHub

  Tech Stack

Python, TensorFlow, Keras, OpenCV, Django, HTML, CSS, MySQL

  Working

  • Encoder Network:Merges a secret message with a cover image to generate a stego-image using CNN layers.
  • Decoder Network:Recovers the hidden message from the stego-image using trained CNN layers.
  • The model uses one-hot encoding for messages and CNNs to learn embedding patterns.

 Features

  • User-friendly Django interface for encoding and decoding.
  • Robust against image compression, resizing, and noise.
  • High payload capacity with negligible visual distortion.
  • Integration with MySQL for admin authentication and tracking.

 Result

Achieved >90% accuracy in message recovery even after image degradation. Stego-images remained visually indistinguishable from cover images. Successfully handled various test cases like different formats, noise addition, and JPEG compression.

  Applications

  • Secure communication in military and corporate fields.
  • Digital watermarking and copyright protection.
  • Confidential file transfer in untrusted environments.

 Movie Recommendation System

A Content-Based Movie Recommendation System that suggests movies similar to a user's selected movie. It uses metadata like genres, keywords, cast, crew, and movie overviews to compute similarity between films.

GitHub   Go Live

  Tech Stack

  • Frontend/UI: Streamlit
  • Backend: Python
  • Libraries: Pandas, NumPy, Scikit-learn, Pickle, Requests, TMDB API
  • Model: Content-Based Filtering

 Features

  • Movie Selection Dropdown-Users can select a movie from a list of titles.
  • Content-Based Filtering-Recommends movies based on cosine similarity of content vectors (genre, cast, keywords, etc.).
  • Poster Display-Each recommended movie is shown with its official movie poster fetched dynamically from the TMDB API.
  • Similarity Score Calculation-Uses TF-IDF Vectorizer or CountVectorizer to convert text metadata into numeric vectors.
  • Preprocessed Movie Dataset-A cleaned dataset with features like title, genre, cast, overview, and tags is used to ensure high-quality results.
  • Efficient Search & Retrieval-Cosine similarity matrix is used to retrieve top N similar movies in milliseconds.

 Result

Provides fast and visually appealing movie recommendations, Demonstrates understanding of NLP techniques and content-based recommender systems Deployed as an interactive web app using Streamlit
